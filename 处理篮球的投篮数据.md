将分数形式的投篮数据转化成进球数和投篮数
========================================================

目前来看，各大数据网站总是将某一个球员的投篮情况，写成如下形式
>Kobe 两分球12/30；

碰到这种情况，我当时一筹莫展，先后请教过周老师和统计之都，周老师告诉我用正则表达式的语言，将投中的次数和投篮的次数分别抓取出来，于是，我将目标转向正则表达式，这个是早晚得学的方面，但是最近还是不太想看。于是，[统计之都](http://cos.name/cn/topic/107748)的***itellin***给出了一个简单的结果：


```r
strsplit("12/30", split = "/")
```

```
## [[1]]
## [1] "12" "30"
## 
```

于是瞬间问题就解决了！！
